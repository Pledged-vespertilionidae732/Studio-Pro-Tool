## ❓ Troubleshooting

Most issues stem from outdated system drivers. If the software closes unexpectedly, check these common fixes.

*   Display Drivers: Update your graphics card drivers through the manufacturer website. Standard Windows updates often miss the latest driver versions for video editing.
*   File Permissions: Ensure you run the installer from a folder where your account has full read and write access.
*   Disk Space: If the program stops during export, check if your target drive has enough room for the final video file. 4K files require significant disk space.
*   Settings Reset: If the interface does not appear correctly, you can delete the configuration folder in your AppData directory to restore default settings.
